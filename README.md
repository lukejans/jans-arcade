### about

Terminal themed arcade to showcase various games I've built while learning html, css, and javascript throughout the [odin project](https://www.theodinproject.com/). This terminal website is primarly compatible with desktop and mobile but with limited functionality as the mobile keyboard takes up much of the screen real estate so I decided to go without commands for mobile and a more button oriented design. Aside from that this website brings the terminal style to some classic games.
